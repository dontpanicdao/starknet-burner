## Examples

This directory provides examples of various setups with and without the
@starknet/burner. Below is a short description of the projects:

- [static](./static) is a simple `.html` file that loads @starknet/burner from
  the `unpkg`/cloudflare CDN
- [vanilla](./vanilla) is a simple vanilla javascript setup that
  @starknet/burner from npmjs.com
- [get-starknet](./get-starknet) is an example of using @starknet/burner with
  get-starknet
- [starknet-react](./starknet-react) is an example of using @starknet/burner 
  with starknet-react
- [starknet-js-v4](./starknet-js-v4) is a document and a set of scripts that
  describes some of the impacts of moving to starknet-js v4 from v3
